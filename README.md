# Bond_Pricing_LGE_SIRD

This repository contains the R code used for the tables, figures, and simulation results reported in the thesis  
**Stochastic Logistic Growth and SIRD Epidemic Modelling in the Context of Pandemic Bond Pricing**.

The code includes:
- reproduction of the benchmark LGE results,
- implementation of the stochastic SIRD--OU framework,
- generation of thesis tables and figures.

The main R scripts are located in the `Bond_Price_LGE_SIRD_Github` folder.

This repository is intended to support the accessibility and reproducibility of the thesis computations.
